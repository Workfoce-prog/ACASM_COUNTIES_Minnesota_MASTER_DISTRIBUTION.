ACASM – Adaptive Complexity-Adjusted Staffing Model
Minnesota County & Statewide Workforce Planning System

Package contents:
1) ACASM_Minnesota_ALL_DEMO.xlsx (prefilled synthetic demo data)
   - Includes embedded sheets: Recommendations, Data_Dictionary
2) ACASM_MN_Streamlit_App_ALL.zip (Streamlit app)

Quick start:
- Excel: open ACASM_Minnesota_ALL_DEMO.xlsx → State_Dashboard / County_Dashboard
- Recommendations + data dictionary are inside the workbook.
- Streamlit: unzip ACASM_MN_Streamlit_App_ALL.zip → pip install -r requirements.txt → streamlit run app.py
- Tableau: connect to ACASM_Minnesota_ALL_DEMO.xlsx → Tableau_Export sheet
